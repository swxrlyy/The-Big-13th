🌐 Cloud Sync Setup Instructions (Firebase)

To enable live, real-time database sync across multiple devices (e.g. from guests at home or across the city), follow these simple steps to configure Firebase:

  1. Go to the Firebase Console and log in with your Google account.
  2. Click Add Project, name it (e.g., "birthday-portal"), and disable Google Analytics (optional).
  3. Once ready, click the Web icon (</>) to register your web application.
  4. Create a database: Click Realtime Database in the left sidebar, click Create Database, select your location, and start in locked mode (or test mode).
  5. Navigate to database Rules tab and set both .read and .write to true (so guests can read/write without authentication): 
    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
  6. Go to Project Settings (gear icon next to Project Overview) and copy the values from the firebaseConfig object.
  7. Open this BDAY.html file in any text editor, locate the firebaseConfig object around line 890, paste your keys, and save.

Note: If no Firebase configuration is entered, the portal automatically falls back to storing data locally inside each user's browser (LocalStorage).

Welcome to the Portal!

We are excited to celebrate with you! Let's take a quick interactive tour to show you exactly how the buttons work so you can customize your experience.

📱

1. Switch Dashboard Layouts

Click these buttons to toggle layouts! **Grid View** shows all widgets at once. **Sidebar View** lets you view each widget individually from a clean selector sidebar (perfect for mobile screens!).

👥

2. See Who's Coming

Click this **Who's Coming tab button** to open the guest list directory, see confirmed attendance statistics, and view other guests' active status updates!

🗺️

3. Get Route Directions

Click the **Get Route Maps button** inside the Location widget to quickly load Google Maps and find travel routes to the gazebo celebrations in Edinburgh.

⚙️

4. Access Settings Options

Click this **Settings tab button** to adjust accessibility sizes, choose custom typography styles, select color themes, or reset your account data.

🛠️

5. Turn on More Widgets

We only display 3 widgets by default to keep it clean. But you can add more! Click the **Open Layout Builder button** to toggle and drag checklist items, hype metrics, and secret mission profiles into your dashboard.
